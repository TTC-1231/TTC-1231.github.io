Overall Framework

Our Test-Time Correction (TTC) framework provides a training-free approach to suppress error accumulation in distilled autoregressive video diffusion models. TTC leverages the key insight that the diffusion sampling trajectory exhibits a natural phase transition: high noise levels determine global structure, while low-noise steps refine appearance details. We apply correction only during the appearance refinement stage, after the global structure has stabilized, using the initial frame as a stable reference anchor. The path-wise correction process operates in three phases: (1) Reference-guided correction via re-noising and denoising with initial frame context, (2) Re-noising the corrected prediction back to the current noise level with fresh Gaussian samples, and (3) Re-denoising using the original evolving context to ensure smooth temporal integration. This approach avoids abrupt state transitions by working entirely within valid diffusion transitions, achieving results comparable to resource-intensive training-based methods with negligible overhead.
